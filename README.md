# TaskFlow

## Descrição
Gerenciador de tarefas colaborativo com autenticação, CRUD e dashboard.

### Tecnologias
- Backend: Node.js, Express, MongoDB, JWT
- Frontend: React, Vite, TailwindCSS

### Como rodar
1. Instale dependências em backend e frontend.
2. Configure .env com MONGO_URI e JWT_SECRET.
3. Execute `npm start` em ambos.
